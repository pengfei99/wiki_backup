====== Liste des Machines Virtuelles ======

Les machines ont un utilisateur **bioaster** dont le mot de passe se trouve dans la base de donnée de mots de passe.

Le compte **root** est accessible en utilisant ''sudo -s''.

Pour ajouter une machine à une des listes ci-dessous, il suffit de créer une page dans le namespace ''/vms/vmware'' ou ''/vms/openstack''. Pour ce faire il suffit de taper le nom complet directement dans la barre d'adresse du navigateur, ex : ''https://ccbiowiki.in2p3.fr/vms/openstack/ma_machine_virtuelle''.
