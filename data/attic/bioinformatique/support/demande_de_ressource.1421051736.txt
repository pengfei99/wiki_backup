====== Pour demander des ressources... ======
Vous avez besoin : 
  * de plus mémoire vive, de plus stockage ou de plus processeur pour vos calculs;
  * de déployer une base de données;
  * d'une machine virtuelle;
  * de déployer une service connecté sur le réseau;
  * de compiler un logiciel.

====== Ou pour demander de l'aide... ======
Vous avez besoin d'aide pour :
  * versionner votre code avec [[bioinformatique:outils:bazaar|Bazaar]];
  * créer un actualiser votre projet dans la forge;
  * soumettre un job sur la ferme de calcul;
  * soumettre un job sur le cloud;
  * vous connectez à vos ressources.

====== Il faut déposer un ticket ======
 Vous pouvez déposer un ticket sur la file BIOASTER du service de support du CC-IN2P3 !

  - Allez sur la page d'accueil sur service support : https://cc-usersupport.in2p3.fr
  - <wrap important>Si c'est votre première visite, cliquez sur [[https://cc-usersupport.in2p3.fr/otrs/customer.pl#Signup|"Sign up now"]] pour créer un compte.</wrap>
      * Adresse électronique : votre adresse bioaster.org.
      * Nom de groupe : bioaster.
      * Nom de laboratoire : BIOASTER.
      * AFSlogon (optionnel) : votre login pour connecter à ccage.in2p3.fr (si vous en possédez un).
  - Connectez-vous et cliquez sur nouveau ticket
  - Remplissez votre demande en fixant la priorité et en sélectionnant BIOASTER dans le champ "Projet"
