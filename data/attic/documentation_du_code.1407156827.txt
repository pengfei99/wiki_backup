====== Utilisation de Doxygen ======

Par défaut, le style de documentation [[http://fr.wikipedia.org/wiki/Doxygen|Doxygen]] peut être suivi pour documenter du code. \\ Doxygen est compatible avec plusieurs langages de programmation C, C++, PHP, Python, PERL, Fortran, R (avec le package [[http://roxygen.org/|Roxygen]]),...\\

Le code documenté peut ensuite servire à générer une doc au format HTML, LaTeX, pdf, docx, etc. [[http://xerces.apache.org/xerces-c/apiDocs-3/classes.html|Exemple]] de documentation HTML générée à partir de Doxygen.
