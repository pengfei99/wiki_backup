===== Move =====
Pour déplacer un espace de nom (c-à-d un répertoire) :
  - Aller sur la page d'entré du répertoire concerné. par exemple, aller sur ''pt6/start'' pour déplacer toutes les pages de ''pt6/''
  - Aller dans Admin/Additional Plugins/Move...
  - Aller dans move namespace 
  - Sélectionner '':[Root namespace]'' dans ''Select new namespace''
  - Ecrire l'espace de destination dans ''New namespace name''. Par exemple ut6 pour déplacer ''root:pt6'' vers ''root:ut6''
  - Soumettre
