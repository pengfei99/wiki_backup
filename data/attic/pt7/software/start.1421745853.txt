=== OpenMS ===

[[http://open-ms.sourceforge.net|OpenMS]] is an cross-platform open-source software C++ library for LC/MS data management and analyses ([[http://ftp.mi.fu-berlin.de/OpenMS/doc-1.9-official/html/index.html|Doc]]).
It comes with ready-to-use tools for 
  * proteomics and metabolomics data analysis ([[http://ftp.mi.fu-berlin.de/OpenMS/doc-1.9-official/html/TOPP_documentation.html|TOPPTools]]) and 
  * 2D and 3D visualization([[http://ftp.mi.fu-berlin.de/OpenMS/release-documentation/html/TOPPView_introduction.html|TOPPView]]).

=== ProteoWizard ===

[[http://proteowizard.sourceforge.net|ProteoWizard]] Library and Tools are a set of modular and extensible open-source, cross-platform tools and software libraries that facilitate proteomics data analysis. 

  * [[http://proteowizard.sourceforge.net/tools/msconvert.html|msconvert]] is a command line tool for converting between various file formats.

==== R ====

  * [[http://www.bioconductor.org|Bioconductor]] is a collection of R packages for bioinformatics
  * [[http://www.bioconductor.org/packages/release/bioc/html/xcms.html|XCMS]]is a Bioconductor-based package for LC-MS, tandem MS/MS data analysis
