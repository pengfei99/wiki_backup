====== Wiki BIOASTER ======

Ce wiki est un espace collaboratif et conviviale où sont regroupées les documentations techniques de **BIOASTER** ainsi que de nombreuses informations.

<WRAP info round>
Pour pouvoir éditer le [[http://fr.wikipedia.org/wiki/Wiki|wiki]] il est nécessaire d'obtenir un [[http://fr.wikipedia.org/wiki/Certificat_%C3%A9lectronique|certificat client]].\\
La procédure est décrite dans ce document : {{ut6:procedure_d_acces_au_wiki.pdf|procedure_d_acces_au_wiki.pdf}} (original au format{{ut6:procedure_d_acces_au_wiki.odt|ODT}}).

Si vous rencontrez des messages d'avertissement en accédant au wiki, suivez [[securite:certificats_de_confiance|cette procédure]] afin d'installer les certificats de sécurité du **CNRS**.
</WRAP>

===== Unités technologiques =====

Les différentes pages de plateaux rassemblent les informations techniques spécifiques à chaque unité : [[pt1|UT1]], [[ut2|UT2]], [[pt3|UT3]], [[pt5|UT5]], [[ut6|UT6]]... 
Ces liens se trouvent dans le menu **navigation** à gauche de la page.

Une page est dédiée aux informations concernant les différents [[instruments|instruments]] disponibles sur les différents plateaux.

===== Employés =====

Les [[employes|pages personnelles]] des employés sont automatiquement listées sur la page ''/employes''. Un lien vers votre page personnelle est aussi disponible en haut à droite de toute les pages.
