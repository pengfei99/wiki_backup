====== Deploy VM via Hyper V ======

===== Steps to deploy VM based on an exported Hyper V vm =====

Follow the GUI of Hyper V to create the VM. Then follow the below steps to config it

==== 1. Change network adapter config ====



<code bash>
# login as root
su -

# edit the /etc/network/interfaces file
nano /etc/network/interfaces

# add the following lines
# This file describes the network interfaces available on your system
# and how to activate them. For more information, see interfaces(5).

source /etc/network/interfaces.d/*

# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface
allow-hotplug eth0
iface eth0 inet static
        address 10.50.5.65/24
        gateway 10.50.0.1
        # dns-* options are implemented by the resolvconf package, if installed
        dns-nameservers 10.50.0.10 8.8.8.8


# save and exit nano

# restart the networking service
systemctl restart networking

# activate the eth0 adaptor
ifup eth0

# check the new IP address 
ip a
</code>

==== 2. Update the system package ====

<code bash>
sudo apt update

sudo apt upgrade
</code>

==== 3. Add user to sudoer list ====

<code bash>
# login as root
su -

# change sudoer list
visudo

# add following line
pliu    ALL=(ALL:ALL) NOPASSWD:ALL
</code>
