====== Gnome desktop configuration ======

To make the newly installed centos 8 more user friendly. We need to install the gnome teak and dash to panel

please visit https://linuxhint.com/tweaking_gnome_desktop_centos8/

if you have problem to find tweak in the software search, use the following command:

<code>
# install epel-repo
sudo dnf install epel-realease

# install tweaks
sudo dnf install gnome-tweaks
</code>
